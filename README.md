# qa_java
Проектное задание 6-го спринта курса "Автоматизатор тестирования на Java" на образовательной платформе ЯндексПрактикум.

#### Автор: Ливадин Александр

### Используемые в проекте технологии:

* Java 11.0.23
* Apache 3.9.8
* JUnit 4.13.1
* Mockito 3.12.4
* Jacoco 0.8.7

### Команда для запуска 
`mvn clean test`
